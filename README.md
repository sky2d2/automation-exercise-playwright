## Project Summary

This project is an automated testing framework built with **Playwright** and **TypeScript** using the **Page Object Model (POM)** design pattern. The framework automates comprehensive user flows on the QABrains Practice Site (https://practice.qabrains.com), covering authentication, form validation, e-commerce, and booking workflows.

## Application Under Test

**Website:** QABrains Practice Site  
**URL:** https://practice.qabrains.com

The site provides multiple testing modules including:
- User Authentication (Login, Registration, Forgot Password)
- Form Submission with validation
- Drag and Drop interactions
- E-commerce flows (Add to cart, Checkout)
- Booking workflows

## Automated User Flows

This framework includes automation for the following 8 user flows:

1. **Login/Logout** - Valid and invalid credential scenarios
2. **User Registration** - New user signup with validation
3. **Forgot Password** - Password recovery flow
4. **Form Validation** - Dynamic form submission with error handling
5. **Drag and Drop** - Interactive UI element manipulation
6. **Add to Cart** - E-commerce product selection and cart management
7. **Checkout Flow** - Complete purchase workflow
8. **Booking Workflow** - Multi-step booking process

## Technologies & Tools

- **Playwright** v1.x - Cross-browser automation framework
- **TypeScript** - Type-safe test development
- **Node.js** - Runtime environment
- **Page Object Model (POM)** - Design pattern for maintainable test code
- **GitHub Actions** - CI/CD integration
